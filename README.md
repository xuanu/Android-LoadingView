# Android-LoadingView
progress 简单的无限旋转的进度条
***  
  自定义一个Loading效果，无限旋转。
  可以自定义两个参数
  ```
  <attr name="border_color" format="color"></attr>//颜色
  <attr name="border_width" format="dimension"></attr>//宽度
  ```
    
- 如果宽度>半径，宽度=半径
